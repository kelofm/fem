Construct a Lagrange polynomial on the provided nodes.

The constructed polynomial is of minimal degree that satisfies the following criteria:

  • evaluates to 1 at the base node
  • vanishes at all other nodes
    Parameters
    p_nodeBeginiterator to the first node.
    p_nodeEnditerator past the last node.
    i_baseindex of the base node within the provided node range (the polynomial evaluates to 1 at this node and vanishes at the rest).
