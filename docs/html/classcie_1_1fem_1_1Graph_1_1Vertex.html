Detailed Description

template<class TVertexData, class TEdgeData>
class cie::fem::Graph< TVertexData, TEdgeData >::Vertex

Vertex type of Graph.

This vertex implementation stores the IDs of the edges originating from- or ending at it, as well as additional data associated with it via TVertexData. Features and storage for the extra data can be disabled without overhead at compile time by setting TVertexData to void.
