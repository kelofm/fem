template<class TVertexData , class TEdgeData >
Ref<Edge> cie::fem::Graph< TVertexData, TEdgeData >::insert ( Ref< const Edge >  rEdge,
bool  overwrite = false 
)

Copy a new Edge into the Graph.

Parameters
rEdgenew edge to be copied into the graph.
overwritein case of ID clashes, this argument controls whether to replace the existing edge with the provided one, or do nothing.

If an edge with an identical ID already exists in the graph, two things can happen depending on the value of overwrite. If overwrite is false (default), then the existing edge remains in the graph and the new one is discarded. If overwrite is true, first the existing edge is erased (eraseEdge), then the new one is inserted.

If the newly inserted edge connects vertices that are not yet part of the graph, default constructed ones are inserted.

See also
eraseEdge
