Detailed Description

template<concepts::Numeric TValue, unsigned Dimension>
class cie::fem::maths::OrthogonalScaleTransform< TValue, Dimension >

Class representing independent scaling along orthogonal coordinate axes.

Uniquely defines a mapping between axis-aligned hyperrectangles in D-dimensional space, that have their base vertices at the origin.
