Detailed Description

template<class TVertexData, class TEdgeData>
class cie::fem::Graph< TVertexData, TEdgeData >::Edge

Edge type of Graph.

This edge implementation stores the IDs of its source- and target vertices. If the edge is directed, it points from the source to the target vertex. Additionally, it also stores extra data associated with the edge, which can be disabled without overhead at compile time by setting TEdgeData to void.
