Detailed Description

template<concepts::Numeric TValue, unsigned Dimension>
class cie::fem::maths::ScaleTranslateTransform< TValue, Dimension >

Class representing independent scaling along orthogonal coordinate axes, followed by a translation.

Uniquely defines a mapping between axis-aligned hyperrectangles in D-dimensional space.
