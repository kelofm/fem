Detailed Description

Hierarchic basis functions as proposed by Szabó & Babuska For easier assembly, the first and last functions are nodal ones, the others vanish at the boundaries.

Constructor arguments: polynomialOrder=1 : polynomial order of the basis to compute the number of functions dtype=np.float64 : storage and function return data types
