Collect ansatz functions that don't vanish on boundaries.

Each boundary is scanned at sample points constructed from the cartesian product of the input sample points, and all ansatz functions that have at least one non-zero value at one of those points are considered to require connectivity on that boundary. An input functor is called with each BoundaryID - ansatz function index pair exactly once.

Parameters

r_functorFunctor that gets called with each BoundaryID and non-vanishing ansatz function index.

p_sampleEndPtr past the last sample node.
