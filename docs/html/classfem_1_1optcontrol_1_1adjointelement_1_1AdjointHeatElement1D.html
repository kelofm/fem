Detailed Description

Element type for AdjointModel The coefficients (capacity, conductivity) don't depend on the solution directly, but on the position, so integration must be carried out accordingly.

The constructor expects a NonlinearHeatElement1D and a forward solution, from which the coefficients are constructed. An extra integration target (nonSymmetricStffness) is added.
